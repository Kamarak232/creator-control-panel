# Style Consistency Prompt System

This document outlines a systematic approach to ensure that 70 images match the exact reference style. The iterative prompt system involves various steps that guide the user towards achieving consistency across all images. 

## Steps for Ensuring Style Consistency
1. **Reference Evaluation**  
   Evaluate the reference images. Identify the core attributes that define the style (e.g., color scheme, composition, texture).

2. **Image Comparison**  
   Compare each image with the reference style. Note inconsistencies in style attributes.

3. **Feedback Gathering**  
   Collect feedback from team members on style perception. Look for common areas of agreement and disagreement.

4. **Adjustments**  
   Based on the feedback, iterate on the images, making necessary adjustments to align them closer to the reference style.

5. **Final Review**  
   Prepare for a final review of all images after adjustments. Ensure that all team members verify consistency.

6. **Documentation**  
   Document changes made during the process for future reference. This helps in maintaining style continuity in future projects.

## Conclusion
The iterative prompt system enables careful evaluation and adjustment of images to achieve style consistency, essential for cohesive visual communication. Maintain open communication with the team throughout the process to ensure alignment.
